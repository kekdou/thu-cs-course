## 基础知识

计算机底层采用 **补码** 的形式储存数字，因为不认识 "+", "-" 号，只能识别 "0" 和 "1"  

假设使用 8 位(8-bit) 整数演示：
- **原码**：最高位为符号位（0正1负），其余位表示数值
  - $+5 \to 0000 0101$
  - $-5 \to 1000 0101$
- **反码**：正数不变，负数将原码除符号位外按位取反
  - $+5 \to 0000 0101$
  - $-5 \to 1111 1010$
- **补码**：正数不变，负数 = 反码 + 1
  - $+5 \to 0000 0101$
  - $-5 \to 1111 1011$

所以对于负数的情况，位运算要谨慎，不要误以为是原码，差距很大   

值得注意的是 **-1 的补码是 1111 1111**  

补码的设计是为了用一套加法器电路完成所有算数运算，例如 5 - 5 = 5 + (-5) = 0000 0101 + 1111 1011 = 1 0000 0000 = 0000 0000 (第九位溢出丢弃)

## 位运算

概述

|符号|描述|规则|
|---|---|---|
|`&`|与|两个位都为1时，结果才为1|
|`\|`|或|两个位都为0时，结果才为0|
|`^`|异或|两个位相同为0，相异为1|
|`~`|取反|0变1，1变0|
|`<<`|左移|各二进位全部左移若干位，高位丢弃，低位补0|
|`>>`|右移|各二进位全部右移若干位，高位补0或符号位补齐|

一些基本用途：
1. 判断奇偶性

```C++
bool is_Odd(int n) {
    if (n & 1) {
        return 1;
    } else {
        return 0;
    }
}
```

2. 快速计算 x 乘除 2 的 n 次幂

```C++
int res = x << n; // x * 2^n
int res = x >> n; // x / 2^n
```

3. 计算 x 对 2 的 n 次幂取模

```C++
int res = x & ((1 << n) - 1) // 保留二进制下后 n 位
```

4. 无分支绝对值计算  

对于 32 位整数，如果是负数，右移 31 位会得到 -1（全 1）；如果是正数，会得到 0（全 0）

```C++
int fastAbs(int n) {
    int mask = n >> 31;
    return (n + mask) ^ mask;
    // mask = 0 时，n ^ 0 = n  
    // mask = -1 时，(n - 1) ^ -1 即减 1 再按位取反
}
```

5. 判断使是否为 2 的幂

```C++
bool isPowerOfTwo(int n) {
    return n > 0 && (n & (n - 1)) == 0;
}
```

6. 无临时变量交换

值得注意的是 $n ^ n = 0$ 以及 $n ^ 0 = n$  

```C++
a ^= b;
b ^= a;
a ^= b;
```

7. 著名的“快速平方根倒数”  

```C++
i = 0x5f3759df - (i >> 1)
```

由于位运算的优先级较低，在使用过程中记得带括号  

## 快速幂  

```C++
int quickPower(int base, int pow) {
    int res = 1;
    while (pow > 0) {
        if (pow & 1) {
            res *= base
        }
        base *= base;
        pow >>= 1;
    }
    return res;
}
```

实现逻辑可以用递归的方式理解，采用分治思想 

```C++
int quickPower(int base, int pow) {
    if (pow == 0) {
        return 1;
    } else if (pow & 1) {
        return quickPower(base, pow - 1) * base;
    } else {
        int temp = quickPower(base, pow >> 1);
        return temp * temp;
    }
}
```

注意此处的 `temp` 是有必要的，如果写成 `quickPower(a, n >> 1) * quickPower(a, n >> 1)`，会计算两次 $a^{n / 2}$ 算法速度显著降低
