## 光标控制

在 C++ 的文件流中存有两个独立的光标  
- `g`(get): 输入（读写）操作
- `p`(put): 输出（写入）操作

|类型|ifstream|ofstream|描述|
|---|---|---|---|
|获取位置|tellg()|tellp()|返回当前光标相对于文件起始位置的偏移量|
|设置位置|seekg()|seekp()|将光标移动到指定位置|

seekg 和 seekp 有两种重载方式
1. 绝对定位

```C++
file.seekg(n) // 跳转到从文件开头算起第 n 个字节
```

2. 相对定位

```C++
file.seekg(offset, direction);
```

- `offset`: 偏移量，正数向后，负数向前
- `direction`: 基准位置，有三个预定义常量

|常量|说明|
|---|---|
|std::ios::beg|开头|
|std::ios::cur|当前|
|std::ios::end|末尾|


## 常见应用

### 获取文件大小

```C++
ifstream file("example", ios::binary);
if (!filr.is_open()) {
    cerr << "Error";
    return 1;
}
file.seekg(0, ios::end);         // 移动到末尾
streamsize size = file.tellg();  // 文件总字节数
file.seekg(0, ios::beg);         // 移回开头进行后续操作
```
