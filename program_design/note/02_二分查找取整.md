## 二分查找

对于单调数列的查找常用二分查找

```c++
int BinarySearch(vector<int> arr, int val) {
    int n = arr.size();
    int l = 0, r = n - 1;
    while (l < n) {
        mid = l + (r - l + 1) / 2;
        if (arr[mid] <= val) {
            l = mid;
        } else {
            r = mid - 1;
        }
    }
    return arr[l];
}
```

这是对于单调非递减数列 `arr` 找出其中最大的不大于 `val` 的项实现方式  

值得注意的是  

```c++
    mid = l + (r - l + 1) / 2;
    if (check()) l = mid;
    else r = mid - 1;
```

- `mid` 采用 l + (r - l + 1) / 2 和 (r + l + 1) / 2 数值上相同，但如果 r 与 l 较大时，前种写法先计算 r - l 可以有效避免数据溢出，因此推荐前种    
- 此外关于取 ceil 和 floor，如果 check() 成立时要求 l = mid，即左边界靠近右边界，则需要 mid 靠近右侧，用 ceil 上取等，否则陷入死循环
- 同理，如果 check() 成立时要求右边界靠近左边界，则 mid 靠近左侧，用 floor 下取等
