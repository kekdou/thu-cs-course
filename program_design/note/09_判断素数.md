## 判断素数

最简单的是筛法，时间复杂度为 $O(\sqrt{n})$  

```C++
bool isPrime(int num) {
    if (num < 2) {
        return 0;
    }
    for (int i = 2; i <= num / i; i++) {  // 采用 num / i 防止溢出
        if (num % i == 0) {
            return 0;
        }
    }
    return 1;
}
```



但是在一些频繁需要检查大素数的情况下，建议制作一份素数表  
比如在 素数和(`sum_prime`) 中可能涉及第 50000 位素数，频繁使用筛法时间消耗大

因此使用埃氏筛-埃拉托斯特尼筛法

```C++
const int MAX_RANGE = 1e6;  // 预估需要处理的范围

std::vector<int> prime;     // 储存素数
bool is_prime[MAX_RANGE];   // 储存是否为素数

void sieve() {
    std::fill(is_prime, is_prime + MAX_RANGE, 1);
    is_prime[0] = is_prime[1] = 0;
    for (int i = 2; i * i < MAX_RANGE; i++) {
        if (is_prime[i]) {
            for (int j = i * i; j < MAX_RANGE; j += i) {
                is_prime[j] = 0;
            }
        }
    }
    for (int i = 2; i < MAX_RANGE; i++) {
        if (is_prime[i]) {
            prime.push_back(i);
        }
    }
}
```

值得注意的是，在两个 for 循环中，外层只需计算 $\sqrt(MAX_RANGE)$ 以内的部分，因为大于的合数会被内层循环筛掉，素数不用处理  
内层循环同样只需从大于 $i * i$ 的部分开始，对任意一个合数 $m$ 考虑素因子分解 $p_1 * p_2 * ... * p_n$ 会大于 $p_1 * p_1$ 因此会被 $n * p_1$ 筛掉
