## 随机数生成

现代 C++ 生成随机数用 `<random>` 库，将随机数分为引擎和分布  

```c++
#include <iostream>
#include <random>

int main() {
    // 1. get seed
    random_device rd;
    // 2. initialize engine
    mt19937 eng(rd());
    // 3. define distribution
    uniform_int_distribution<int> disN(1, 100);
    uniform_real_distribution<double> disR(1.0, 100.0);
    int r1 = disN(eng);
    double r2 = disR(eng);
    return 0; 
}
```

- **种子：**
  - `random_device rd`: 请求操作系统提供一个随机数，如果你的电脑硬件支持，它生成的是真随机数
  - `time(0)`: 如果 `random_device` 在某些旧系统上不可用，可以用传统的 `time(0)` 作为备份种子
- **引擎：**
  - `mt19937`: 生成 32 位随机数
  - `mt19937_64`: 生成 64 位随机数（范围超过 $2^{31}-1$ 时使用）
- **分布：**
  - `uniform_int_distribution<int> dis(min, max)`: 均匀整数分布
  - `uniform_real_distribution<double> dis(0.0, 1.0)`: 均匀浮点数分布
  - 此外还有正态分布，伯努利分布等

引擎生成一个或多个原始随机数并交给分布，分布根据内部算法（如映射、截断或数学变换），将这些原始位转换成符合你要求的数  

## 对拍

将现有程序和确保正确的程序的结果进行比对，以检测离谱情况

- `data.cpp`: 生成随机测例
- `check.cpp`: 自动运行程序并比对运行结果

对于 hw2_equation 示例

```c++
// data.cpp
#include <iostream>
#include <random>
#include <ctime>

using namespace std;

const double MAX = 10000.0;
const double MIN = -100000.0;

int main() {
    mt19937 rng(time(0)); 
    uniform_real_distribution<double> dist(MIN, MAX);
    uniform_int_distribution<int> type_dist(1, 10);
    int test_type = type_dist(rng);
    double a, b, c, d;
    if (test_type == 1) {
        a = c = dist(rng);
        b = d = dist(rng);
    } else if (test_type == 2) {
        a = c = dist(rng);
        b = dist(rng);
        d = dist(rng);
        while( b == d) {
            d = dist(rng);
        }
    } else {
        a = dist(rng);
        b = dist(rng);
        c = dist(rng);
        d = dist(rng);
        while (a == c) {
            c = dist(rng);
        }
    }
    cout << a << " " << b << endl;
    cout << c << " " << d << endl;
    return 0;
}
```

```c++
// check.cpp
#include <iostream>
#include <windows.h>
#include <cstdlib>
#include <ctime>

using namespace std;

int main() {
    int count = 0;
    while (count < 1000) {
        count++;
        system("data.exe > input.txt");
        system("equation.exe < input.txt > equation.txt");
        system("test.exe < input.txt > test.txt");
        if (system("fc /b equation.txt test.txt > nul 2>&1")) {
            cout << "Wrong #" << count << endl;
            break;
        } else {
            cout << "Passed #" << count << endl;
        }
    }
    return 0;
}
```
